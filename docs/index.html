Monalisa

Responsive, scalable, minimalist and beautiful CSS framework

only 8.4KB minified and gzipped

"The Supreme law of art is the representation of the beautiful"

  • Responsive

    Monalisa is prepared for all devices. The grid system is easily configurable according to the resolution and design settings.

  • Scalable

    Designed for any size of project; remember: every project can (and probable will) became bigger. Monalisa is modular, well documented and easy to maintain.

  • Mobile First

    Mobile first, this is the right way; every one should start as simple as possible. Monalisa is light and clean to all devices.
